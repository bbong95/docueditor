# docueditor - AI 기반 고성능 프레젠테이션 에디터 🚀

[English](./README.md) | [한국어](./README.ko.md) | [简体中文](./README.zh.md) | [日本語](./README.ja.md) | [العربية](./README.ar.md) | [Français](./README.fr.md) | [Español](./README.es.md)

`docueditor`는 최첨단 AI 기술을 결합하여 고품질 프레젠테이션을 제작하고 편집할 수 있는 강력한 도구입니다.

## 주요 기능
- **AI 기반 슬라이드 생성**: 텍스트나 이미지를 프레젠테이션 슬라이드로 즉시 변환합니다.
- **3D 인터랙티브 에이전트**: 실시간으로 협업을 돕는 지능형 3D 에이전트 시스템을 제공합니다.
- **고해상도 렌더링**: 캔버스 기반의 정밀한 텍스트 및 이미지 배치 시스템을 통해 전문적인 결과물을 보장합니다.
- **다양한 포맷 지원**: PDF 분석 및 PPTX 내보내기 기능을 지원합니다.

## 기술 스택
- **프론트엔드**: React + TypeScript + Vite
- **3D 렌더링**: React Three Fiber + Three.js
- **상태 관리**: Zustand
- **AI 엔진**: Google Gemini API
- **테스트**: Vitest

## 시작하기
```bash
npm install
npm run dev
```
