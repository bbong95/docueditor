`docueditor` is a state-of-the-art presentation editing platform that leverages advanced AI to bridge the gap between static documents and dynamic, high-fidelity presentations.

## âœ¨ Core Features
- **AI Slide Generation**: Instantly transform raw text or images into structured presentation slides using Gemini 3.0.
- **3D Interactive Agents**: Intelligent 3D avatars (Lego style) that assist you in real-time during the editing process.
- **Precision Rendering**: Canvas-based text and image placement system ensuring pixel-perfect layout alignment.
- **Enterprise Grade Exports**: Support for PDF analysis and high-fidelity PPTX exports.
- **Performance Optimized**: Optimized bundle size (~1.88MB) for fast global delivery.
