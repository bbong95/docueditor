# Project Profile: docueditor

## Overview
A high-fidelity slide editor and visual documentation generator. Built with React, Zustand, and Lucide. Integrated with Gemini API for OCR and style analysis.

## Current Technical Stack
- Frontend: React 18, TypeScript, Vite
- State Management: Zustand (useEditorStore)
- PDF Processing: pdfjs-dist
- Diagramming: Rough.js, Mermaid
- Styling: Vanilla CSS (MUI/Premium style)

## Major Features
1. PDF/Image Upload & Conversion
2. OCR Text Detection & Styling
3. Visual Documentation Portal (Architect/Analyst views)
4. Slide Export (Image/PDF)
