# docueditor - AI 搭載型高性能プレゼンテーションエディタ 🚀

[English](./README.md) | [한국어](./README.ko.md) | [简体中文](./README.zh.md) | [日本語](./README.ja.md) | [العربية](./README.ar.md) | [Français](./README.fr.md) | [Español](./README.es.md)

`docueditor` は、最先端の AI 技術を融合させ、高品質なプレゼンテーションを作成・編集できる強力なツールです。

## 主な機能
- **AI スライド生成**：テキストや画像を即座にプレゼンテーションスライドへ変換します。
- **3D インタラクティブエージェント**：リアルタイムで共同作業を支援するインテリジェントな 3D エージェントシステムを搭載。
- **高解像度レンダリング**：キャンバスベースの精密な配置システムにより、プロフェッショナルな品質を実現。
- **多様な形式に対応**：PDF 解析および PPTX 書き出しをサポート。

## 技術スタック
- **フロントエンド**：React + TypeScript + Vite
- **3D レンダリング**：React Three Fiber + Three.js
- **状態管理**：Zustand
- **AI エンジン**：Google Gemini API
- **テスト**：Vitest

## セットアップ
```bash
npm install
npm run dev
```
