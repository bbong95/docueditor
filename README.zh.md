# docueditor - AI 驱动的高性能演示文稿编辑器 🚀

[English](./README.md) | [한국어](./README.ko.md) | [简体中文](./README.zh.md) | [日本語](./README.ja.md) | [العربية](./README.ar.md) | [Français](./README.fr.md) | [Español](./README.es.md)

`docueditor` 是一款结合了尖端 AI 技术的高性能演示文稿制作和编辑工具。

## 主要功能
- **AI 幻灯片生成**：将文字或图像即时转换为演示文稿。
- **3D 交互式代理**：提供实时辅助协作的智能 3D 代理系统。
- **高分辨率渲染**：通过精密的画布渲染技术，确保专业的视觉效果。
- **多格式支持**：兼容 PDF 解析及 PPTX 导出。

## 技术栈
- **前端**：React + TypeScript + Vite
- **3D 渲染**：React Three Fiber + Three.js
- **状态管理**：Zustand
- **AI 引擎**：Google Gemini API
- **测试**：Vitest

## 快速开始
```bash
npm install
npm run dev
```
