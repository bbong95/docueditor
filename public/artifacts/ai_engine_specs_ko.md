# 🧠 AI 엔진 상세 명세 (DocuEditor v2.0)

## 1. 핵심 엔진: Gemini 3.0 Pro
DocuEditor의 심장은 구글의 **Gemini 3.0 Pro**입니다. 고차원적인 추론 작업과 복잡한 텍스트 추출에 사용됩니다.

### 1.1 모델 구성
- **모델명**: `gemini-3.0-pro-exp`
- **온도(Temperature)**: `0.2` (정밀한 추출을 위해 낮게 설정)
- **TopK**: `40`
- **출력 토큰 제한**: `8192`

### 1.2 기능 매트릭스
| 기능 | 역할 | 구현 방식 |
| :--- | :--- | :--- |
| **OCR** | 복잡한 시각 정보에서 텍스트 추출 | 멀티모달 비전(Vision) 역량 |
| **번역** | 추출된 텍스트 번역 | 제로샷(Zero-shot) 번역 |
| **논리** | 레이아웃 및 의미 블록 분석 | 공간 추론(Spatial Reasoning) |
| **코드** | UI 컴포넌트 생성 | 코드 생성 (React/TS) |

---

## 2. 폴백 엔진: Tesseract.js
API 할당량 초과 시에도 "무중단" 서비스를 보장하고, "비용 제로(Zero-Cost)" 운용을 지원하기 위한 엔진입니다.

- **역할**: 클라이언트 사이드 OCR 폴백(Fallback).
- **트리거**: `429 Too Many Requests` 에러 발생 시 자동 전환.
- **성능**: 속도가 빠르으나, 복잡한 레이아웃에서의 정확도는 다소 낮음.

---

## 3. 생성형 아트: Imagen 3 (via Gemini)
플레이스홀더 이미지 생성 및 시각적 자산 개선에 사용됩니다.

- **모델**: `imagen-3.0-generate-001`
- **용도**: UI 목업 생성, 아이콘(폴백), 다이아그램 배경 일러스트 생성.

## 4. 에이전트 페르소나 정의 (Constitution)
각 AI 에이전트는 고유한 시스템 지침("Constitution")을 부여받아 행동합니다.

### 4.1 The Architect (Lead)
> "당신은 프로젝트 리더입니다. 요청을 분석하고, 계획을 수립하며, 업무를 위임하십시오. 'Task.md'를 진실 공급원(Source of Truth)으로 관리하십시오."

### 4.2 The Designer (UI/UX)
> "당신은 세계적인 수준의 UI/UX 디자이너입니다. 심미성, 패딩(여백), 글래스모피즘, 사용자 즐거움에 집중하십시오. Tailwind CSS를 효과적으로 사용하십시오."

### 4.3 The Developer (Core)
> "당신은 시니어 풀스택 개발자입니다. 깨끗하고 효율적이며 타입 안전성이 보장된 TypeScript 코드를 작성하십시오. TDD 원칙을 준수하십시오."

### 4.4 The Analyst (QA)
> "당신은 꼼꼼한 QA 엔지니어입니다. 모든 산출물을 요구사항과 대조하여 검증하십시오. 리포트를 생성하고 예외 케이스를 찾아내십시오."
